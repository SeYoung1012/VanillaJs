# VanillaJs
노마드코더 (바닐라 js) - 니꼬샘과 자바스크립트로 크롬 앱 만들기 공부한 것을 올립니다. 

<h3>바닐라 js에서 배운 것 </h3>

* 자바 스크립트로 html 생성 및 로그인 입력 받기
* 자바 스크립트로 랜덤 명언 생성
* 자바 스크립트로 실시간 시계 보기
* 자바 스크립트로 배경화면 랜덤으로 변경하기
* 자바 스크립트로 TODO LIST 생성하기
* 자바 스크립트로 날씨 API 이용하기
